Milestone 2 – Customer Base Audit

This project analyzes customer purchasing behavior using a customer base audit approach. The objective is to evaluate how revenue, profit, and transactions are distributed across different customer segments.

Project Overview

The analysis divides customers into deciles based on their purchasing activity. This allows identification of the most valuable customers and comparison of performance across segments.

The Excel analysis includes:
	•	Decile analysis of customers
	•	Distribution of transactions, revenue, and profit
	•	Average revenue and profit per customer
	•	Average order frequency
	•	Order value and margin analysis

Tools Used
	•	Microsoft Excel
	•	Data tabulation and statistical calculations
	•	Graphical visualization

Skills Demonstrated

This project demonstrates the ability to organize customer data, perform statistical calculations, analyze customer value distribution, and present results in a clear analytical structure suitable for business decision-making.
