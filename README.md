# Quantitative and Digital Skills Portfolio – Omirzak Aibat

This project presents a financial portfolio analysis comparing Mercedes-Benz Group AG (MBG.DE) and Toyota Motor Corporation (TM) using historical data from Yahoo Finance.

## Project Overview
The analysis includes:
- Collection and cleaning of financial data
- Calculation of weekly returns
- Descriptive statistics (mean, variance, standard deviation, z-scores)
- Risk assessment using Sharpe Ratio and F-test
- Regression analysis with the S&P 500 index

## Tools Used
- Microsoft Excel
- Yahoo Finance data

## Files
- Portfolio_Toyota_Mercedes_Omirzak_Aibat.xlsx – Excel file with all calculations and analysis
- Synopsis.pdf – Short explanation of the portfolio and methods used

This project demonstrates the ability to analyze financial data, apply statistical methods, and interpret investment performance.
